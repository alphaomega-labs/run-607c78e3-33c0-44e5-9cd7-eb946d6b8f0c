# Limitations Appendix

- Full compilation backend was approximated in confirmatory real checks by clingo configuration modes; this preserves executable evidence but weakens direct compile-vs-enum attribution.
- Several benchmark families are workspace-generated and therefore support stress testing, not external benchmark replication claims.
- Counterexample detection target in cache audits was not met and remains an explicit negative result.
- Router envelope-violation targets were only met in bounded real-check tables, not across all simulation ablations.
